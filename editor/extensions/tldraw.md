# tldraw

Infinite canvas whiteboard inside VS Code — sketch diagrams, wireframes, and flowcharts without leaving the editor. Files saved as `.tldr`.

- **Extension ID**: `tldraw-org.tldraw-vscode`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=tldraw-org.tldraw-vscode

## Install

```bash
code --install-extension tldraw-org.tldraw-vscode
```

## Usage

- `Cmd+Shift+P` → **tldraw: New** — create a new drawing
- Open any `.tldr` file to edit it in the canvas
- Supports shapes, arrows, text, freehand drawing, and sticky notes
