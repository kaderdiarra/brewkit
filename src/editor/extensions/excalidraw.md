# Excalidraw

Whiteboard and diagramming tool inside VS Code — sketch diagrams, wireframes, and flowcharts without leaving the editor. Files saved as `.excalidraw`.

- **Extension ID**: `pomdtr.excalidraw-editor`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=pomdtr.excalidraw-editor

## Install

```bash
code --install-extension pomdtr.excalidraw-editor
```

## Usage

- `Cmd+Shift+P` → **Excalidraw: Create new drawing** — create a new drawing
- Open any `.excalidraw` file to edit it in the canvas
- Supports shapes, arrows, text, freehand drawing, and libraries
- Export to PNG/SVG directly from the editor
